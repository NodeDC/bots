Tickets

Beginner

The beginner ticket is for anyone who has little or no hardware, JavaScript or Node experience. You will learn the basics of writing a Node program to control hardware. You’ll blink an LED, listen for input from buttons and different sensors and become familiar with the basics for interfacing your computer programs with the physical world. This ticket includes an Arduino compatible Uno clone and an assortment of basic hardware components.

Cost TBD

Beginner with Bot

Are you a beginner who wants to go a little further? This ticket is for you. It covers all the same ground and includes all the same components as the Beginner ticket, but you’ll also get a Sumobot Jr. kit to assemble and control with Node. At the end of the day we will have a big Sumobot battle bracket.

Cost TBD

Intermediate with Bot

Are you new to hardware but comfortable with Node? This ticket is for you. We’ll cover all the basics of controlling LED’s and listening for input just like the beginners, but instead of using an Arduino compatible board, you’ll be using a Tessel 2. The Tessel 2 is a single board computer that runs Linux and node and does not require a physical connection to a host computer. It’s pretty awesome!

Cost TBD

Hacker

Come on down and spend time hacking on your personal project, showing off your existing nodebot creations and helping the beginners get up and running.
