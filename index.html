Welcome!

Come be a part of an International NodeBots Day hosted here in Houston, TX. July 30th is the NodeBots day for 2016, which is an annual event where JavaScript programmers gather together to hack on, or learn to hack on, hardware. For additional information on the event, please visit: nodebots.io.

The event is organized by several houston community leaders including Node.js Houston, Houston Robotics Club, and Houston IoT Meetup.

For additional questions, reach out to us on the NodeJS Houston Slack group under the #nodebots channel. Register an account here

Tickets

Main Nodebots Track

Are you new to NodeBot and want to build your own bot? This ticket is for you. You will learn the basics of writing a Node program to control hardware. You’ll blink an LED, listen for input from buttons and different sensors and become familiar with the basics for interfacing your computer programs with the physical world. This ticket includes an Arduino compatible Uno clone, an assortment of basic hardware components and a a Sumobot Jr. kit o assemble and control with Node. At the end of the day we will have a big Sumobot battle bracket.

$50

Beginner Kit, without bot

The beginner ticket is for anyone who has little or no hardware, JavaScript or Node experience. You will learn the basics of writing a Node program to control hardware. You’ll blink an LED, listen for input from buttons and different sensors and become familiar with the basics for interfacing your computer programs with the physical world. This ticket includes an Arduino compatible Uno clone and an assortment of basic hardware components.

$25

Advanced NodeBots Kit

Are you new to hardware but comfortable with Node? This ticket is for you. We’ll cover all the basics of controlling LED’s and listening for input just like the beginners, but instead of using an Arduino compatible board, you’ll be using a Tessel 2. The Tessel 2 is a single board computer that runs Linux and node and does not require a physical connection to a host computer. It’s pretty awesome!

FAQs

  • What is a NodeBot?

    • Nodebots are hardware controlled with JavaScript. This includes robots and IoT connected devices.

  • What will I learn?

    • You'll learn how to control hardware using Node.js and the johnny-five library.

  • Is this a Hackathon?

    • No. We dont have prizes. There are no stakes.

  • Do I have to have any experience with Robots or Programming?

    • No, you don't. You do need to bring a laptop computer with NodeJS installed.

  • Can I bring my kids?

    • Yes, kids are welcome. If you will be sharing hardware, please purchase a hacker ticket to cover their lunch.

Resources

Want to get started with JavaScript and hardware? Here are some of our favorite resources:

  • node-ardx - tutorials for using using an Arduino with node and various components
  • nodebots workshop - a tutorial in the nodeschool 'workshopper' format that requires a bit of terminal knowledge to get started.
  • sumobot tutorial - slightly dated video tutorial on assembling and programming a sumobot
  • Johnny-Five - a high level JavaScript library for working with hardware
